# SBSE + LLM LAB

Este repositório é uma coleção aberta de laboratórios práticos para aprender **SBSE (Search-Based Software Engineering)** utilizando **Modelos de Linguagem Grande (LLM/IA)**, como Gemini, Llama, Mistral, Deepseek-coder, via API ou local (Ollama).

**Público alvo:** Estudantes, desenvolvedores, professores e curiosos interessados em otimização aplicada à engenharia de software e integração com IA generativa.

**Aqui você encontra:**
- Scripts de benchmark e experimentação
- Exemplos de integração SBSE+IA
- Código pronto para rodar local ou em cloud (Colab)
- Ferramentas para comparar diferentes LLMs na formulação de problemas de otimização

## Como usar

- Clone este repositório
- Escolha o script para seu ambiente:
  - Local com Ollama: `workshop_ollama_local.py`
  - Google Colab + Gemini: `workshop_gemini_colab.ipynb`
- Leia o início dos scripts para configurar seu modelo ou chave

**Sinta-se à vontade para enviar PRs, adaptar para outros modelos de LLM ou contribuir com estudos sobre SBSE + IA!**

# SBSE + LLM LAB

This repository is an open collection of practical laboratories to learn about **SBSE (Search-Based Software Engineering)** using **Large Language Models (LLM/AI)** such as Gemini, Llama, Mistral, Deepseek-coder, via API or locally (Ollama).

**Target audience:** Students, developers, teachers and anyone interested in optimization applied to software engineering and generative AI integration.

**Here you will find:**
- Benchmark and experimentation scripts
- SBSE+AI integration examples
- Ready-to-run code for local or cloud (Colab)
- Tools to compare different LLMs on optimization problem formulation

## How to use

- Clone this repository
- Choose the script for your environment:
  - Locally with Ollama: `workshop_ollama_local.py`
  - Google Colab + Gemini: `workshop_gemini_colab.ipynb`
- Read the beginning of the scripts to configure your model or API key

**Feel free to send PRs, adapt to other LLM models, or contribute with studies about SBSE + AI!**
